# docs/

Source for [agentguides.io](https://agentguides.io). Built with Jekyll + [just-the-docs](https://just-the-docs.com/) via GitHub Pages.

## Local preview

```sh
cd docs
mise install            # one-time: installs Ruby 3.3 if not already present
bundle install
bundle exec jekyll serve
```

Serves at <http://localhost:4000>.
